CSS (Cascading Style Sheets)

CSS stands for Cascading Style Sheets

CSS is used for looks and feel of html

Types of CSS

1. Inline CSS

this is used for apply css property in same line of tag with the help of style tag

2. Internal CSS

internal css write code in head section with the help of stype tag. selectors are used in internal css

3. External CSS

external css write code in Another file with the help of ".css" extension and link it with html file with the help of link tag of href property.

Note

Inline css is powerful in all css

SELECTORS

Selectors are the used to select the element in html and apply css property on it.

Types of Selectors

1. ID Selector

ID selectors are used for particular element symbol="#"

2. Class Selector

Class selectors are used for group of element symbol="."

3. Tag Selector

Tag selectors are used for all element of same tag symbol="tag name"

4. Compound Selector

Compound selectors are used for more than one selector symbol="selector1, selector2"

5. Pseudo Selector

Pseudo selectors are used for particular state of element symbol=":,::"

6. Universal Selector

Universal selectors are used for all element symbol="*"
