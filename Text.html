FORM TAG

form tag is used for creating a form with some properties.

1. action: action tells performance of form with page

2. method: method is used to specify the HTTP method to be used when submitting the form data. It can be either "get" or "post".

3. enctype: enctype is used to specify the encoding type of the form data when it is submitted. It can be "application/x-www-form-urlencoded", "multipart/form-data", or "text/plain".
